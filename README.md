# Social-Media-Dashboard---Power-BI-Project
This project is a Power BI Social Media Analytics Dashboard built to visualize and analyze social media performance metrics using interactive charts, KPI cards, and trend analysis.

The dashboard provides insights into:

Social media engagement
Weekly performance trends
KPI monitoring
Comparative analytics
Interactive data exploration
